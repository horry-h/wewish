# 设计稿描述 - 心之解惑/AI答案之书

## 视觉风格定位
- **主题色系**: 深夜疗愈模式(Deep Dark Mode) + 莫兰迪色系点缀
- **背景**: 深邃渐变(#0a0e27 → #1a1e3a),微妙星空粒子效果
- **字体**: 思源宋体/华文楷体(营造古典书卷气息)
- **特效**: 柔和光晕、呼吸动画、流光边框

### 视觉细节
- **每日一签卡片**: 
  - 白色半透明背景(rgba(255,255,255,0.08))
  - 灯笼图标做微微摇摆动画
  - 右上角显示日期(农历)
  
- **问题分类按钮**:

- **核心交互按钮**:
  - 直径260rpx的圆形
  - 渐变色:#d4a574 → #f4d5a6(金色系)
  - 3层阴影营造悬浮感
  - 中心水晶球图标(SVG,带光泽)

## 页面二:交互页 (Interaction / Seeking)

### 动态效果描述
此页面为全屏沉浸式动画页,用户长按首页按钮后进入。

```
┌────────────────────────────────┐
│                                │
│                                │
│        [书页翻动动画]           │  ← 主视觉区
│                                │     3D透视效果
│       📖 → 📖 → 📖             │     书页快速翻过(10张/秒)
│                                │     半透明叠加
│      (粒子汇聚特效)            │
│                                │
│   ════════════════════         │  ← 进度条(流光效果)
│                                │     金色渐变填充
│                                │
│      "答案即将显现..."          │  ← 提示文字(淡入淡出)
│                                │     随机切换禅语
│                                │
│                                │
│   [请保持长按]                 │  ← 底部提示(脉冲闪烁)
│                                │
└────────────────────────────────┘
```

### 技术要点
- **书页动画**: CSS 3D transform + rotateY
- **粒子系统**: Canvas绘制50-100个光点,向中心汇聚
- **震动反馈**: 每0.3秒调用`wx.vibrateShort({type:'medium'})`
- **音效**: 白噪音(流水/风铃)循环播放,音量渐强
- **松开检测**: 立即停止动画,淡出过渡到结果页

### 禅语库(随机显示)
- "万物皆有裂痕,那是光照进来的地方"
- "此刻的犹豫,正在为未来铺路"
- "答案从不迟到,只是时机未至"
- "内心的声音,最为真实"

## 全局交互规范

### 震动反馈(Haptic Feedback)
| 场景 | 震动类型 | API调用 |
|-----|---------|---------|
| 按钮点击 | 轻震 | `wx.vibrateShort({type:'light'})` |
| 类别选择 | 中震 | `wx.vibrateShort({type:'medium'})` |
| 长按开始 | 持续轻震 | 循环调用 |
| 答案显示 | 重震 | `wx.vibrateShort({type:'heavy'})` |

### 音效设计
| 场景 | 音效类型 | 文件名 |
|-----|---------|--------|
| 页面切换 | 空灵铃声 | `bell.mp3` |
| 书页翻动 | 纸张沙沙声 | `page-turn.mp3` |
| 长按背景 | 白噪音(流水) | `water-bg.mp3` |
| 答案揭晓 | 清脆"叮" | `reveal.mp3` |

### 动画时长规范
- 微交互(按钮点击):150ms
- 页面过渡:400ms cubic-bezier(0.4, 0, 0.2, 1)
- 书页翻动:100ms/帧
- 打字机效果:50ms/字

## 适配说明
- **屏幕尺寸**: 以iPhone 12 Pro(390x844)为基准
- **安全区**: 考虑刘海屏/底部横条,使用`env(safe-area-inset-*)`
- **暗黑模式**: 目前仅支持深色主题
- **可访问性**: 所有操作按钮尺寸≥88rpx,对比度≥4.5:1

> **设计理念**: 通过极简的视觉语言和富有仪式感的交互,让用户在短暂的"冥想时刻"获得心理慰藉,而非单纯的随机答案。
